# Portfolio Numérique – Introduction aux Systèmes d'Information

Portfolio web réalisé dans le cadre du cours Introduction aux Systèmes d'Information en MIAGE.  
Il regroupe l'ensemble des travaux, notes de cours, documents et livrables produits durant le module.

---

## Contenu du portfolio

Le portfolio est organisé en 7 rubriques accessibles via une navbar interactive :

| # | Rubrique | Contenu |
|---|----------|---------|
| 1 | Travaux demandés | TDs (études de cas, PGI, Odoo), TP Gantt, schémas MERISE, diagrammes UML |
| 2 | Notes de cours | SI vs PGI, processus de gestion, SFG/SFD, Use Case |
| 3 | Documents SI | Cours complémentaires |
| 4 | Certifications | — |
| 5 | Projet final | Cahier des charges, faisabilité, SFG, SFD, dossier technique, rapport de tests, diagrammes |
| 6 | Diaporama | — |
| 7 | Ressources | Exemples de cahiers des charges |

---

## Technologies utilisées

- HTML5 / CSS3 / JavaScript (Vanilla)
- Ionicons pour les icônes
- Google Fonts (Poppins)
